This is the body of the first element accordion. It is displayed by default until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the devices and hiders using CSS transitions. You can change all of this with your own CSS or override our default variables. It is also worth noting that almost any HTML can be placed in the .accordion-body, although the transition limits the overflow.
